## 🚀 Project Overview
FounderFlow is a full-stack Java web application designed to manage startup ideas and community voting. It focuses on **Data Integrity** and **Database Automation** using advanced DBMS concepts.

## 🛠 Tech Stack
* **Frontend:** JSP, HTML5, CSS3
* **Backend:** Java Servlets (MVC Architecture)
* **Database:** MySQL
* **Tools:** Apache Tomcat, JDBC, NetBeans IDE

## ✨ Key DBMS Features
* **Automated Voting (Triggers):** A MySQL Trigger is used to automatically update the vote count in the `ideas` table whenever a new record is inserted into the `votes` table.
* **3NF Normalization:** The database schema is fully normalized to minimize redundancy.
* **Complex Joins:** Used to fetch real-time trending startup data.
* **Data Security:** Implemented using PreparedStatements to prevent SQL Injection.
